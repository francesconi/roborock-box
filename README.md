# Roborock Garage

## Prerequisites

### Hardware

- Raspberry Pi
- NEMA 17 Stepper Motor
- Stepper Motor Controller
- DRV8825 Stepper Motor Driver

### Software

- [Just](https://github.com/casey/just#installation)

## Deployment

```sh
just deploy
```
